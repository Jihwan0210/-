## 프레임워크(Framework)에 대한 설명으로 옳은 것은?

### 문제
1. 소프트웨어 구성에 필요한 기본 구조를 제공함으로써 재사용이 가능하게 해준다.
2. 소프트웨어 개발 시 구조가 잡혀 있기 때문에 확장이 불가능하다.
3. 소프트웨어 아키텍처(Architecture)와 동일한 개념이다.
4. 모듈화(Modularity)가 불가능하다.

### ✅ 정답
**1번**

### 📌 핵심 정리
- 프레임워크는 소프트웨어 개발을 위한 **기본 구조(뼈대)** 제공
- 공통 기능 재사용 → **생산성·유지보수성 향상**
- **제어의 역전(IoC, Inversion of Control)** 개념 적용

---

## HDLC 프레임 구조 순서

### ✅ 정답
**Flag → Address → Control → Information → FCS → Flag**

### 📌 핵심 정리
- **Flag**: 프레임 시작·종료 표시 (`01111110`)
- **Address**: 송수신 스테이션 주소
- **Control**: 프레임 종류·순서·흐름 제어
- **Information**: 실제 데이터 (선택적)
- **FCS**: 오류 검출 (CRC)
- **Flag**: 프레임 종료
